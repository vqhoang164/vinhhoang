App-Controlled Pet Feeder

App-Controlled Pet Feeder Project

Objective

The project in the Advanced Hardware Design course aims to combine hardware and software to design a practical application. This automatic feeder can dispense dry food for my pet. The features are:

  • System to be controlled manually and remotely using network.
  • Network database for timestamp records.
  • Motor controller to adjust the amount of food dispensed.
Approach

The schematic shows how different systems interface. I use Raspberry Pi 4 as the brain for the feeder. Basically, it controls a stepper motor through GPIO pins and connected to a network cloud database: Google Firebase. A button is connected to the Pi for manual control. Next, I wrote an Android App that sends signals and updates variables on Firebase. The Raspberry keeps monitoring the database and response in real time. It also writes timing records to Firebase, and the App can retrieve and display to the user. The Raspberry code is written in Python.


The feeder is an aftermarket food container. It is modified by connecting the dispense mechanism to a stepper motor. Instead of using DC or Servo motor, stepper motor can offer more precise positioning control to achieve the same number of steps, or the same revolution at each cycle. In addition, speed is not a crucial consideration for this application, but torque is a higher priority. The motor is bipolar, with the 1.8o rotation per step. Therefore, it needs 200 steps to complete a full revolution. The 3.3 V pin cannot provide enough power to the motor. Instead, I use L298N motor-driver module and connect 12V external power to provide full torque. The module has two H-bridges and each one drives the coil of the motor. The enable pins from the module are connected to the GPIO pins of the Raspberry Pi.


The Android App is coded in Android Studio (JavaScript). It has only one segment containing all main activities. The App connects to the network to write variables on Firebase and read strings sent from Raspberry to display timestamps.

What I learned: Python, Android App development, Raspberry Pi, Network interface, Stepper Motor Control
